## General
- Save: `:w`
- Quit: `:q`
- Save + quit: `:wq`
- Undo: `u`
- Redo: `<C-r>`

## Keys
- <leader>: Space
- `C`: Ctrl

## Navigation
- Go to definition (LSP): `gd`
- References: `gr`
- Hover docs: `K`
- Go back: `<C-o>`
- Go forward: `<C-i>`

## Plugins
- Lazy UI: `:Lazy`
- Mason UI: `:Mason`
- Sync plugins: `:Lazy sync 

## Diagnostics / Errors
- Show error under cursor: `gl`
- Next error: `]d`
- Prev error: `[d`
- Location list: `:lopen`
- Quickfix list: `:copen`
